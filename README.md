## Variables

- Variables are used for storing data
- Use the smallest amount of memory possible
- There are 6 main data types

| Data Type | Value                    |
| --------- | ------------------------ |
| int       | Whole numbers            |
| long      | Very large whole numbers |
| double    | Decimals                 |
| String    | Text                     |
| char      | A single character       |
| boolean   | true or false            |
